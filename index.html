Sitio web para una Clinica Odontologica


El sistema permite la gestión de citas de pacientes, asignación de turnos y la administración eficiente de los recursos clínicos. Además, proporciona funcionalidades de control de acceso para distintos roles en la clínica, incluyendo odontólogos, secretarios y administradores del sistema.

Repositorio GitHub: ClinicaOdontologica.git
Tecnologías: Java, POO, JPA (ORM), JSP, HTML, CSS, JavaScript y MySQL

Sistema de gestion de automoviles para una Concesionaria

Sistema para llevar un control completo del inventario de vehículos de una concesionaria. Permite dar de alta, realizar ediciones, visualizar y dar de baja automóviles de manera eficiente y organizada. Ya sea que se estén agregando nuevos modelos al inventario, actualizando información, mostrando detalles a los clientes o retirando vehículos vendidos.

Repositorio GitHub: TiendaAutomoviles.git
Tecnologías: Java, POO, JPA (ORM), y MySQL

Sistema de gestion de clientes para una Peluqueria Canina


Sistema para llevar un control completo de los clientes de una peluquería canina. Permite agregar, editar, visualizar y dar de baja las mascotas registradas de manera simple y organizada. Ya sea que estés registrando nuevos clientes de cuatro patas, actualizando sus datos, viendo detalles sobre sus servicios o retirando mascotas atendidas, esta herramienta te facilita el trabajo en la peluquería canina.

Repositorio GitHub: PeluqueriaCanina.git
Tecnologías: Java, POO, JPA (ORM), y MySQL
